# Analyzing Abusive Content in Indian Vernacular Languages: A Case Study of Marathi and Bangla

Partial implementation accompanying the paper:

**Analyzing Abusive Content in Indian Vernacular Languages: A Case Study of Marathi and Bangla**  
📍 **FIRE 2025 (Forum for Information Retrieval Evaluation), IIT BHU, India**

📄 **Paper:** <https://dl.acm.org/doi/full/10.1145/3777867.3777880>

---

## Overview

This repository contains the datasets and trained models used for multilingual hate speech detection in **Marathi** and **Bangla**. The work investigates classical machine learning, deep learning, and transformer-based approaches for offensive language identification in low-resource Indian languages.


## Datasets

- **MOLD v2** (Marathi Offensive Language Detection)
- **BAD** (Bangla Abusive Language Detection)
- **HASOC Hindi 2020** (Hindi Datset)
